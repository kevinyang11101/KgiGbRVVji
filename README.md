# 医疗设备管理系统 python342

## 项目概述

医疗设备管理系统是基于 Python Django 框架开发的一套应用软件，旨在帮助医疗机构高效管理其设备采购与库存信息。

## 技术栈

- **后端**: Python Django
- **数据库**: MySQL
- **前端**: HTML

## 功能模块

- **用户认证**
  - 登录
  - 注册
  - 个人信息修改
- **采购管理**
  - 采购申请
  - 采购审核
  - 采购通过
- **库存管理**
  - 设备名称设置
  - 设备单价设定
  - 库存信息维护

## 系统角色

- 系统管理员
- 采购管理人员
- 设备使用人员

## 运行环境

- Python 环境配置
- Django 环境搭建
- MySQL 数据库服务

## 部署步骤

1. 配置 Python 环境及 Django 框架
2. 安装依赖包
3. 配置 MySQL 数据库
4. 初始化数据库结构
5. 运行 Django 应用

## 目录结构

```
medical_equipment_system/
│
├── apps/                 # 应用目录
├── db/                   # 数据库相关文件
├── media/                # 媒体文件目录
├── static/               # 静态文件目录
├── templates/            # HTML 模板目录
│
├── manage.py              # 管理脚本
└── requirements.txt       # 依赖列表
```

## 核心亮点

- 使用 Django 框架，提供稳定的后端支持
- 基于 MySQL 数据库，确保数据安全性和完整性
- 简洁的目录结构和清晰的文档，便于维护和二次开发
- 分角色管理，满足不同用户的操作需求，提高工作效率
